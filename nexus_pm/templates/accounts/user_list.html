Quick Reference — User Management
Quick Role Change

Hover over any user's role badge and click the pencil icon. A modal will open — select the new role and confirm. No page reload needed.

Password Reset

Click the icon on any row to set a new password. Users can also change their own password from their profile page.

Deactivate vs Delete

Deactivate blocks login but keeps all data intact. Delete permanently removes the user and all their records.
